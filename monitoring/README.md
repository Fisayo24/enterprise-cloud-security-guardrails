# Monitoring

This module handles logging and monitoring using AWS CloudWatch.

## Architecture

![Monitoring Architecture](../monitoring-architecture.png)

## Architecture Overview

This diagram illustrates the monitoring and logging architecture for the environment. 
AWS CloudWatch is used to centralize logs, enabling visibility into system activity and performance.

Log groups are configured to collect and store logs from various resources, with defined retention policies to support auditing and compliance. This setup ensures efficient monitoring, faster troubleshooting, and improved security posture across the infrastructure.
