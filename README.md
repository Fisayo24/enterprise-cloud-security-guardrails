# Enterprise Cloud Security Guardrails

This project demonstrates how to implement cloud security guardrails using Terraform, AWS, and DevSecOps practices. The goal is to enforce secure infrastructure configurations and automate security checks within a CI/CD pipeline.

## Technologies
- Terraform
- AWS (S3, CloudWatch)
- GitHub Actions
- DevSecOps security automation

## Security Controls
- S3 Public Access Block
- Server-Side Encryption
- Bucket Versioning
- CloudWatch Monitoring
- Infrastructure as Code validation

## Project Purpose
This project showcases practical DevSecOps and Cloud Security engineering skills including infrastructure automation, security guardrails, and CI/CD integration.
